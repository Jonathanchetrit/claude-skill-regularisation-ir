# Référence 05 — Checklist des documents par type de revenu

## Documents UNIVERSELS (tout dossier de régularisation)

| Document | Obligatoire | Usage |
|---|---|---|
| Pièce d'identité | ✓ | Identification contribuable |
| Numéro fiscal (13 chiffres) | ✓ | Obligatoire sur tous les formulaires |
| Avis d'imposition des années régularisées | ✓ | Vérifier déclarations existantes, cotisations payées |
| Adresse du SIP compétent | ✓ | Domicile fiscal |
| Justificatif de domicile | ✓ | Résidence fiscale |
| Acte de naissance | Selon dossier | Vérification état civil complet |
| Livret de famille | Selon dossier | Parts fiscales, enfants à charge |

---

## § A — Comptes bancaires et titres étrangers

| Document | Obligatoire | Usage | À demander à |
|---|---|---|---|
| Relevés de compte annuels (tous comptes étrangers) | ✓ | Mouvements, soldes | Banque / client |
| Activity Statements annuels (courtiers : IBKR, Degiro, etc.) | ✓ | Dividendes, PV, intérêts | Courtier / client |
| IFU (Imprimé Fiscal Unique) si disponible | Souhaitable | Pré-rempli par l'établissement | Établissement payeur |
| Attestation d'ouverture / clôture de compte | ✓ | Formulaire 3916 | Banque |
| Contrats d'ouverture de compte | Souhaitable | Dates exactes | Banque / client |
| Fichier Webstat BCE (taux de change officiels) | ✓ | Conversions EUR | Télécharger banque-france.fr |

---

## § B — Plus-values mobilières

| Document | Obligatoire | Usage | À demander à |
|---|---|---|---|
| Activity Statements / relevés de cessions | ✓ | PV et MV par titre | Courtier |
| Justificatifs d'acquisition des titres cédés | ✓ | Prix de revient (taux jour d'achat) | Courtier / relevés historiques |
| Ordres de passage pour les acquisitions | Souhaitable | Prix exact et date | Courtier |
| Moins-values antérieures reportables | ✓ si pertinent | Imputation sur PV | Déclarations antérieures / DGFiP |
| 2074 antérieurs (si déposés) | Souhaitable | Vérifier cohérence | Client / DGFiP |

> ⚠️ Pour les titres acquis en devises étrangères plusieurs années avant la cession,
> le taux BCE du JOUR d'acquisition est indispensable. Prévoir un délai pour le retrouver.

## § C — Dividendes et intérêts étrangers

| Document | Obligatoire | Usage | À demander à |
|---|---|---|---|
| Activity Statements section Dividends | ✓ | Dividendes bruts + retenues | Courtier |
| Activity Statements section Interest | ✓ | Intérêts créditeurs | Courtier |
| Activity Statements section Withholding Tax | ✓ | Retenues à la source | Courtier |
| IFU étranger équivalent (ex : Form 1099 USA) | Souhaitable | Vérification croisée | Courtier / client |
| Notices 2047 des années concernées | ✓ | Taux crédit d'impôt par pays | impots.gouv.fr ou avocat |

## § D — Actifs numériques (crypto)

| Document | Obligatoire | Usage | À demander à |
|---|---|---|---|
| Historique complet des transactions (CSV) | ✓ | Calcul PV/MV méthode PMP | Plateforme (Coinbase, Binance…) |
| Relevés de toutes les plateformes utilisées | ✓ | Inventaire complet | Client |
| Valeur portefeuille au 01/01 et 31/12 de chaque année | ✓ | Calcul PMP | Plateforme |
| Cours crypto au moment des cessions | ✓ | Valorisation | CoinGecko, Binance, etc. |

## Documents de DÉFENSE (pour demande de remise gracieuse)

| Document | Usage | À demander à |
|---|---|---|
| Certificat médical (psychiatre / médecin traitant) | Cause étrangère — état de santé | Médecin |
| Certificats d'hospitalisation | Période d'incapacité | Hôpital |
| Attestation d'invalidité / AAH | Vulnérabilité | MDPH / CAF |
| Acte de décès (proche) | Circonstance exceptionnelle | Mairie |
| Attestation d'absence à l'étranger | Méconnaissance loi française | Client |
| Capture d'écran incident technique | Impossibilité matérielle déclaration en ligne | Client |
| Tout justificatif de bonne foi | Argumentation générale | Client |

## Matrice de criticité

| Statut | Signification | Action |
|---|---|---|
| ✓ REÇU | Document disponible et analysé | Passer au calcul |
| ⚠ EN ATTENTE | Demandé, pas encore reçu | Relancer client / tiers |
| ✗ BLOQUANT | Calcul impossible sans ce document | Suspendre la phase concernée |
| ~ ESTIMÉ | Montant approché retenu, à confirmer | Mentionner l'hypothèse dans les documents |
| N/A | Non applicable pour ce profil | — |
